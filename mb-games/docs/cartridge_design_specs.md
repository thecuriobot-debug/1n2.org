# Cartridge Design Specs
Project: Bear Market Breakout (Atari 2600) + Mempool Meltdown (NES)
Studio: MAD BITCOINS RETRO GAME LAB

## Bear Market Breakout (Atari 2600)
### Visual Intent
- Mood: aggressive market chaos with bright satoshi highlights.
- Palette: black, deep navy, blood-red, amber, mint green accent.
- Typography: bold monospace, all-caps arcade look.

### Label Content
- Title: `BEAR MARKET BREAKOUT`
- Tagline: `HODL. BREAK FUD.`
- Footer credit: `MAD BITCOINS RETRO GAME LAB`
- Platform badge: `ATARI 2600`

### Print Guidance
- Recommended label art canvas: 700x900 px (master), export 300 DPI print versions.
- Keep 5% bleed area around edges for trimming.
- Use matte stock with light satin laminate.

### Packaging Copy (Short)
- Front: `Smash through red-candle panic and stack sats in the fastest breakout on the blockchain.`
- Back bullets:
  - `2 game modes: Classic + Chaos`
  - `Meme powerups: HODL, LASER, MOON`
  - `Pure score-attack replayability`

## Mempool Meltdown (NES)
### Visual Intent
- Mood: technical urgency, chain stability under pressure.
- Palette: electric blue, white, crimson alert, yellow accent.
- Layout: clean NES-era box hierarchy with art panel + red platform strip.

### Label and Box Content
- Title: `MEMPOOL MELTDOWN`
- Tagline: `SORT TX. CLEAR BLOCKS. SAVE THE CHAIN.`
- Footer credit: `MAD BITCOINS RETRO GAME LAB`
- Platform strip: `NINTENDO ENTERTAINMENT SYSTEM`

### Print Guidance
- Box/front master canvas: 900x1200 px, export with 300 DPI for print proofs.
- Cartridge label master: 700x900 px.
- Preserve 4 mm safe margin for title text and logos.

### Packaging Copy (Short)
- Front: `The mempool is overflowing. Can your node survive the panic?`
- Back bullets:
  - `Endless Marathon + 20 Challenge puzzles`
  - `Chain combos and fee-tier strategy`
  - `Fork Panic boss rounds every 5 stages`
