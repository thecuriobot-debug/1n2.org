# Bitcoin Trains

Train-based Bitcoin price tracker inspired by moving ticker toys.

## What it does

- Polls live BTC/USD price every 15 seconds.
- Keeps the train always chugging.
- Tilts the track and changes train mood based on up/down/flat price direction.
- Uses CoinGecko first with Coinbase fallback.

## Run

Open `index.html` directly in a browser, or serve the folder with any static server.
