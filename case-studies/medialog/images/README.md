## Screenshot Guidelines

- **Size:** 1200px width (desktop), 375px width (mobile)
- **Format:** PNG for UI, JPG for photos
- **Quality:** High resolution (2x for retina)
- **Annotations:** Add arrows/highlights for key features
- **Consistency:** Use same browser/theme for all shots

## Taking Screenshots

### macOS
```bash
# Full screen
Cmd + Shift + 3

# Selected area
Cmd + Shift + 4

# Specific window
Cmd + Shift + 4, then Space
```

### Browser DevTools
1. Open DevTools (F12)
2. Toggle device toolbar (Cmd+Shift+M)
3. Set size to 1200x800 or mobile
4. Take screenshot
