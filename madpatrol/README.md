# Moon Patrol: Bitcoin Battle

Retro browser game inspired by Moon Patrol with Bitcoin-themed levels and bosses.

## Run

1. Open `/Users/thunt/Downloads/CODEX-PROJECT/index.html` in a browser.
2. No install required, no paid APIs, no external libraries.

## Controls

- `Arrow Keys`: Move
- `Up Arrow`: Jump (ground stages)
- `Space`: Shoot (forward + upward + rear shot) / Start from title
- `Esc`: Pause/unpause during gameplay

## Included

- 8 levels with different rulesets:
  - Side scrolling
  - Underwater
  - Flying
  - Top-view shooter
  - Bubble/Tower variants
  - Final boss (Satoshi)
- Bitcoin-themed powerups and background billboards
- Retro HUD, score/lives, stage splash text, title/win/game-over screens
