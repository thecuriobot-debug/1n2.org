Curio-DID

Difference-in-Differences Analysis for Curio Cards · 📄 Research Paper (PDF)
What is this? Curio-DID runs causal inference on Curio Cards NFT data using the Difference-in-Differences method. For each event (marketplace listing, viral tweet, whale buy), we estimate whether it caused a change in prices — not just correlation. The model: Y = β₀ + β₁·Treatment + β₂·Post + β₃·(Treatment×Post) + ε where β₃ is the causal treatment effect.
