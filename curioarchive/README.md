# 🦝 Curio Archive

**The Definitive Encyclopedia of Curio Cards**

A comprehensive, Wikipedia-style archive documenting the first art NFT collection on Ethereum.

## 🌐 Live Site

Visit: **http://localhost:8000/curioarchive/**

## 📚 What's Inside

The Curio Archive is a complete encyclopedia featuring:

### 📇 Complete Card Database
- **All 31 cards** (Cards 1-30 + the legendary 17b misprint)
- Detailed information for each card including:
  - Artist, supply, mint date, release week
  - Full descriptions and historical significance
  - IPFS hashes and technical details
  - Burned/locked token counts

### 🎨 Artist Profiles
- **7 complete artist biographies**:
  - Phneep - Created the Story Set (Cards 1-10)
  - Cryptograffiti - Bitcoin propaganda (Cards 11-13)
  - Cryptopop - Crypto memes (Cards 17-19 + 17b)
  - Robek World - RPG Founders, first animated NFT (Cards 21-23)
  - Daniel Friedman - Hand-drawn patterns (Cards 24-26)
  - Marisol Vengas - Collaborative fine art (Cards 27-29)
  - Thoros of Myr - The mysterious Eclipse artist (Card 30)

### 👥 Founder Information
- Thomas Hunt (Mad Bitcoins) - Conceptualist
- Travis Uhrig - Software Developer  
- Rhett Creighton - Lead Developer

### 📅 Complete Timeline
- From the 2016 Proof of Art show to the 2021 Christie's auction
- 19 major events documented
- Color-coded by importance (Critical, High, Medium)
- Categorized by type (Origin, Launch, Milestone, Notable, etc.)

### 📰 Featured Articles
- **"The Origin Story: From Baseball Cards to Blockchain Art"**
- **"The Great Rediscovery of 2021"**
- **"The Mystery of Card 17b"**
- **"From $60 to $1.27 Million: The Christie's Moment"**
- **"Card #23: The First Animated NFT on Ethereum"**

### 🔍 Advanced Features
- **Full-text search** - Search across all cards, artists, and articles
- **Wikipedia-style navigation** - Clean, familiar interface
- **Hash-based routing** - Direct links to any card, artist, or article
- **Responsive design** - Works on desktop and mobile
- **Infoboxes** - Wikipedia-style information panels
- **Categorized sets** - Cards organized by thematic sets
- **Interactive timeline** - Visual history of the project
- **Quick facts sidebar** - Key information always visible

## 🎨 Design Philosophy

The Curio Archive is designed to look and feel like Wikipedia:

- **Clean typography** - Linux Libertine for headers, system fonts for body
- **Infoboxes** - Right-aligned information panels
- **Neutral color scheme** - Wikipedia's grays and blues
- **Responsive layout** - Main content + sidebar
- **Accessible** - High contrast, semantic HTML
- **Fast** - No external dependencies, pure vanilla JS

## 📖 Usage Examples

### Direct Links
```
http://localhost:8000/curioarchive/#card/1      - View Card #1
http://localhost:8000/curioarchive/#card/17b    - View the 17b misprint
http://localhost:8000/curioarchive/#artist/phneep - View Phneep's profile
http://localhost:8000/curioarchive/#article/origin-story - Read origin story
http://localhost:8000/curioarchive/#timeline    - View timeline
```

### Navigation
- Click any card number to see full details
- Click artist names to see their profile and all their cards
- Use the search bar to find anything
- Navigate between pages using the header menu

### Search
- Type in the search bar (minimum 2 characters)
- Results appear instantly
- Shows card numbers, artist names, and article titles
- Click any result to jump to that page
