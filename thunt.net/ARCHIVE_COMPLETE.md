## 💡 **How It Works:**

### Archive Process:
1. Parse all Net Interest links from SQL
2. Download each webpage (10 sec timeout)
3. Parse HTML for images
4. Download up to 10 images per page
5. Update HTML with local paths
6. Save metadata
7. Track success/failure
8. Generate statistics
9. Create updated page

### Error Handling:
- Timeout after 10 seconds
- Retry with http:// if needed
- Graceful failure tracking
- Rate limiting (1 sec between requests)
- Save partial results
