# MADNINJA
## NES-STYLE ACTION PLATFORMER DESIGN DOCUMENT
### Second Pass - Expanded Production Spec

Inspired by the gameplay and cinematic presentation of Ninja Gaiden (1988 NES video game).

## Project Goal
Build a fast, responsive side-scrolling action platformer in the style of the NES era, starring MadNinja as a cyber-ninja fighting enemies threatening Bitcoin and the open internet.

The game should capture the feel of late-era NES action games:
- fast controls
- challenging enemies
- dramatic cinematic cutscenes
- memorable boss battles
- chiptune soundtrack

Tone mix:
- retro action
- hacker cyberpunk
- Bitcoin memes
- humorous enemies

## Core Gameplay
Genre: 2D side-scrolling action platformer.

Core mechanics:
- run
- jump
- wall cling
- wall jump
- sword attack
- sub weapon attack
- enemy knockback
- platform navigation
- boss battles

Target difficulty: classic NES challenge, difficult but fair.

## Main Character
Hero: MadNinja

## Weapon System

- Cane Sword

Secondary weapons:

- Blockchain Bomb

## Currency System
Collect SATOSHIS for:
- sub weapon ammo
- score
- bonus rewards

Collect 1000 sats: gain extra life.

## Health System
- player health bar

## Items and Powerups

- Lightning Mode: temporary speed boost
