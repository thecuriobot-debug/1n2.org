# Curio Archive Static Site

Markdown-driven static site system for Curio Archive.

## Build

```bash
npm run build
```

Generated output is written to `dist/`.

## Structure

- `content/`: Markdown pages and update entries
- `data/`: Structured JSON datasets
- `templates/`: HTML template modules
- `lib/`: Parsing helpers
- `assets/`: CSS and client-side JS
- `scripts/`: Static generation scripts
- `dist/`: Generated site
